Для корректной установки кистей в блендер следуйте следующим указаниям:

1) Скопируйте содержимое папки OrbBrushes (А именно в скопированном должно быть: папка Icons, папка textures и файл OrbBrushes.blend)

2) Перейдите по пути:
C:\Users\{Имя пользователя}\AppData\Roaming\Blender Foundation\Blender\2.78\config 
для Windows

~/.config/blender/2.78/config
Для Linux

Вставьте в эту папку все скопированные файлы

3) Сделайте чистый запуск белнедера (Именно чистый. Под этим я понимаю обычный запуск блендера, ни в коем случае не делайте запуск какой либо рабочей сцены, обычный чистый запуск блендера для начала новой работы)

4) Нажмите File -> Append

5) Перейдите по пути C:\Users\{Имя пользователя}\AppData\Roaming\Blender Foundation\Blender\2.78\config (Windows)
~/.config/blender/2.78/config (Linux) и нажмите на OrbBrushes.blend

6) В появившихся папках выберите папку Brush

7) Выделите все файлы с пометкой "A| Orb " можно сделать box select (нажмите латинскую B на клавиатуре и выбирете всей файлы)

8) Нажмите Append from Library

9) После чего снова жмите File -> Save Startup File

Всё, блендер должен подхватить все кисти.

Приятного блендинга, товарищи!
